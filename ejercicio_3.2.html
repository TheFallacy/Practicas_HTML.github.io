Ejercicio 2

Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso explique por qué.

  • a)Cualquier tipo input particular de formulario de HTML% debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML.
    Respuesta:Falso, puede haber diferencias dependiendo el navegador que se utilice.
  • b)Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor.
    Respuesta:Falso, el placeholder solo desaparece.
  • c)No es necesario incluir el atributo autofocus en sus formularios
    Respuesta:Verdadero.
  • d)Los nuevos tipos input de HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos inválidos que se envían.
    Respuesta: Verdadero.
  • e)EL tipo input range se valida automáticamente de manera intrínseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los límites de valor mínimo o máximo.
    Respuesta: Verdadero.
  • f) HTML5 cuenta con validicación automática para el tipo input tel.
    Respuesta: Falso, no cuenta con validación.
  • g) Si el usuario introduce un URL con un formato inapropiado en un tipo input url, no se validará. HTML5 no valida si el URL introducido existe en realidad.
    Respuesta:Verdadero.
  • h)EL elemento nav muestra un menú desplegable de hipervínculos.
    Respuesta:Falso, no despliega nada, solo es un elemento que te puede ayudar para agrupar vinculos.
  • i) El elemento header puede usarse sólo una vez por página.
    Respuesta:Falso. puede usarse varias veces.
  • j)Los elementos nav pueden anidarse en un elemento aside.
    Respuesta:Verdadero.
  • k)Podemos usar el elemento brk para evitar dividir palabras en lugares correctos.
    Respuesta:Falso, el elemento que se utiliza para esta tarea es wbr
